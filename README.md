# 👾 About
Simple DLL Injector that uses VirtualAllocEx, CreateRemoteThread, & LoadLibrary.

# ☂️ Usage
1. Clone this repository
2. Change DLL path and target game process
3. Build in `Release | x86`
4. Run while target game is open
5. Enjoy!
